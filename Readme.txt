GuideLine How To Run App On Localhost

1. Open the terminal on your system.
2. Navigate to the folder containing folder "src".
3. If this is the first time you run this app on localhost server, run command "npm i" to install node_modules.
4. To open program in localhost, run command "npm start"
5. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
